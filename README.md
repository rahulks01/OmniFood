# Omnifood
```
A basic frontend project using the HTML5 and CSS3 technologies. 
Omnifood is a fictional startup based on delivering healthy meal-kits.
```
## About Omnifood
```
Omnifood is a tech-driven company focused on promoting healthy eating through AI. 
Our app customizes weekly meal plans based on user preferences. 
We partner with local restaurants to prepare and deliver these meals in selected cities, 
offering convenient monthly subscriptions with options for one or two meals per day.
```
## How to run on your machine

```
- Clone the repository:
     Open the terminal or command prompt in your local computer and prompt the following command
     git clone https://github.com/rahulks01/OmniFood.git

- Open the index.html file in your web browser.
```
